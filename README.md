# sales-performance-dashboard
This project analyzes sales data from 2014 to 2017 to uncover key business insights related to revenue, profitability, customer behavior, and regional performance.  The dashboard helps stakeholders identify profit leakage, optimize product strategy, and improve overall business performance.
